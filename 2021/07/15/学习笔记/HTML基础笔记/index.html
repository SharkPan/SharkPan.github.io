    HTML基础笔记

    发表于 2021-07-15 更新于 2021-09-06 分类于 学习笔记 本文字数： 5.4k 阅读时长 ≈ 5 分钟

    知识点提及

    Web 标准的构成（重点）

    ​ 主要包括结构（structure框架html）、表现（presentation颜色css）、行为（behavior交互js），要互相分离。

    HTML语法规范

    ​ 单标签极少出现

    ​ 包含关系与并列关系（此处是骨架标签，需记）

    ​ “! + tab” 自动生成html骨架
    ! DOCTYPE = document type

    语法

    语义标签

    标题标签h1~h6

    段落标签

    ​ （根据浏览器页面大小，自动换行）

    换行标签

    ​ （强制换行）
    ​ br = barter rabbet

    黑体标签

    ​ （更推荐使用strong标签，因为在CSS里可以做进一步的美化）

    斜体

    ​ （更推荐使用em标签，语义更强）
    ​ em = emphasize

    删除线

    ​ （更推荐使用del标签，语义更强）
    ​ s/strike = strikethrough

    下划线

    ​ （更推荐使用ins标签，语义更强）
    ​ ins = inserted text

    非语义标签

    盒子标签📦

    ​ （div是分区，一个div自己占据一整行；span是跨距，一行里可以有很多span。简言之，div是大盒子，span是小盒子）
    ​ div = division

    图像标签🌆

    ​ （注意是单标签，可改width，height「通常width和height里只改一个，不然会让图片变形」，border「通过CSS修改样式」）
    ​ src = source

    ​ （当图片无法显示时，以文字替换）

    ​ （当鼠标移动到图片上，显示提示文本）

    ​ （绝对路径的符号 \ ，相对路径的符号 / ）

    超链接🔗

    1. 外部链接
      （href的开头必须是 http:// 或 https://）
      （target是指用什么方式打开：_self 为默认值，即在本页打开；_blank 为在新窗口打开）
      a = anchor href = hypertext reference
    1. 内部链接

    ​ （空链接的 href 值为 “#”）

    ​ （下载链接的 href 值为 .exe 或 .zip 等压缩包文件名称）

    ​ （以图片为载体的超链接）

    ​ 内部链接中一个比较特殊的形式：锚点链接
    ​ （立刻跳转到本页面的指定位置，注意设立标签需要 # ）

    表格标签 ⊞

    ​ 表格不是用来布局页面的，而是用来展示数据的。
    ​ tr = table row td = table data (cell) th = table header

    ​ ⚠️熟悉属性单词，之后CSS会使用
    ​ （align为对齐方式属性，值含”left”, “center”, “right”；border为边框属性，值含”1”, “”；cellpadding为单元格与内容之间的距离，默认”1”；cellspacing为单元格之间的距离，默认”2”；width 与 height也都可用）

    ​ （用上表格结构标签thead和tbody）

    ​ （合并单元格rowspan跨行「竖着」，colspan跨列「横着」；使其等于”合并单元格的个数”；row最上面单元格为目标单元格，col最左边单元格为目标单元格）
    ​ colspan = “column span”

    列表标签🧾

    ​ 列表最大的优点在于他的整齐、整洁、有序，所以适合做页面布局，共分为无序列表、有序列表、自定义列表。

    • ⚠️重点：无序列表（ul里只能包含li，但li里可以放任何元素）

    ​ ul = “unordered list” li = “list item”

    • 有序列表（ol里只能包含li，但li里可以放任何元素）

    ​ ol = “ordered list”

    • ⚠️重点：自定义列表（dl里只能包含dt和dd，可出现多个dt，dt和dd是并列关系

    ​ dl = “definition list” dt = “definition term” dd = “definition discription”

    表单标签📝

    ​ 使用表单的目的是为了收集用户信息，主要是由表单域（表单的区域）、表单控件（表单元素）、提示信息（文字信息）这3个部分组成。

    • 表单域 form（为了将信息送至后端服务器；action属性值为url地址「服务器程序地址」，method属性值为”GET”、”POST”「提交方式」，name属性值为名称「用于区分同一个页面中的多个表单域）
    • 表单控件（表单元素）

    ​ input标签（name和value是几乎每个表单元素都有的属性值，主要给后台人员使用）用于采集用户信息，type属性有如下：

    属性值 描述
    button 定义可点击按钮(多数情况下,用于通过JavaScript启动脚本)。
    checkbox 复选框（可设置默认选中**checked=”checked”**）
    file 定义输入字段和“浏览”按钮，供文件上传。
    hidden 定义隐藏的输入字段。
    image 定义图像形式的提交按钮。
    password 隐藏密码
    radio 单选按钮（⚠️必须是同一个name属性才可以实现多选一；可设置默认选中**checked=”checked”**）
    reset 定义重置按钮。重置按钮会清除表单中的所有数据。
    submit 定义重置按钮。重置按钮会清除表单中的所有数据。
    text 单行的输入字段，用户可在其中输入文本。默认宽度为20个字符，可通过修改maxlength来调整（不常用）。
    ✩
