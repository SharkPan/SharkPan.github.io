前端自学万人血书

发表于 2021-09-02 更新于 2021-09-03 分类于 课外摘录 本文字数： 1.1k 阅读时长 ≈ 1 分钟

前端自学万人血书

书籍推荐

1.HTML - Head First HTML 与 CSS （有趣配图）
- HTML5权威指南 （全面性高，内容较多）
教程 https://www.w3school.com.cn/h.asp
菜鸟教程
2.CSS - CSS权威指南（当工具书）
- CSS揭秘（实践教程，启发性）
教程同样 w3school&菜鸟
3.JavaScript - 你不知道的JavaScript（对深度概念讲解）
- JavaScript DOM编程艺术（入门）
- JavaScript 高级程序设计（红宝书，权威入门书籍，全面)
- JavaScript权威指南（全面性高）
教程 菜鸟&现代JavaScript教程（https://zh.javascript.info/)
4.ES - ECMAScript6入门教程
- 深入理解ES6（系统地介绍）
5.TS -英文文档 https://www.typescriptlang.org/docs
-中文文档 https://www.tslang.cn/docs/home.html
-深入理解TypeScript
6.数据结构与算法
-学习JavaScript数据结构与算法「第三版」
7.设计模式
-JavaScript设计模式（灵活运用，进阶时需要）
8.网络协议
-图解TCP/IP，图解HTTP（非常有趣）
-HTTP权威指南（不知包括http协议，扩大知识面与进阶时需要）
9.NodeJS
-中文文档 http://nodejs.cn/api/
-Node.js实战（硬要找教材，可以看看）
-深入浅出Node.js（进阶，想深入了解可看）
10.框架
-首推官方文档
Vue.js中文官方文档：https://vuejs.bootcss.com/guide/
React.js官方文档：https://reactjs.org/docs/getting-started.html
React.js小书：https://github.com/huzidaha/react-naive-book
Webpack中文官方文档：https://www.webpackjs.com/concepts/
Sass中文文档：https://sass.bootcss.com/documentation
Less中文文档：http://lesscss.cn/usage/
Stylus中文文档：https://stylus.bootcss.com/
11.性能
-Web性能权威指南（TCP/IP内容先熟悉后，进阶时食用）
